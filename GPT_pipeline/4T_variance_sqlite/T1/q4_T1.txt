To calculate the number of samples that belong to healthy patients, we can use the metadata table and filter by 'disease' column to get the samples from healthy patients.

Here is the SQL query:
SELECT COUNT(DISTINCT sample_id)
FROM metadata
WHERE metadata_column_name = 'disease'
AND metadata_variable_content = 'healthy';
